Privacy Policy for PageXray - Web Performance Analyzer

Data Collection and Usage

PageXray is designed with privacy in mind and operates entirely within your browser:

  1. Local Processing: All performance data is processed and stored locally in your browser.
  2. No Data Transmission: We do not collect, transmit, or store any of your browsing data on external servers.
  3. Types of Data Collected Locally:
    • Performance metrics (page load times, resource timings)
    • Resource information (size, type, load time)
    • Web Vitals measurements
    • Technical page information (URL, title)

Data Storage

  • All data is stored using Chrome's local storage API
  • Data remains on your device and is never transmitted externally
  • You can clear all stored data at any time through the extension's settings

Third-Party Services

PageXray does not use any third-party services or analytics.

Changes to This Policy

Any changes to our privacy policy will be updated with the extension and noted in the changelog.

Contact

If you have questions about this privacy policy, please contact us at pagexray@example.com.
